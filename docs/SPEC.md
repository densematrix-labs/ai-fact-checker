# AI Fact Checker / 谣言溯源器 — Mini Spec

## 目标
帮助用户验证网络信息的真实性，通过 AI 分析提供可信度评估和潜在来源追踪。

## 核心功能
1. **事实核查**：用户输入一段声明/新闻，AI 分析其可信度（高/中/低）
2. **论据提取**：识别声明中的关键论点，逐条评估
3. **来源追踪**：推测该信息可能的传播路径和原始来源类型
4. **对比分析**：与已知事实/常识进行对比，指出矛盾点
5. **Disclaimer**：明确标注"AI 分析仅供参考，不代表事实判定"

## 技术方案
- **前端**：React + Vite (TypeScript) + TailwindCSS
- **后端**：Python FastAPI
- **AI 调用**：通过 llm-proxy.densematrix.ai
- **部署**：Docker → langsheng
- **端口**：Frontend 30040, Backend 30041

## 用户流程
1. 用户访问首页
2. 粘贴/输入要核查的声明或新闻
3. 点击"核查"按钮
4. 等待 AI 分析（显示 loading）
5. 显示结果：
   - 可信度评分（高/中/低 + 百分比）
   - 关键论点分解
   - 可疑之处标注
   - 可能的来源类型分析
6. 用户可保存/分享结果
